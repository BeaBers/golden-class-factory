# 📐 شرح هيكل المشروع

## المبادئ الأساسية

1. **التنظيم الرقمي**: كل مجلد يبدأ برقم لترتيب القراءة
2. **الفصل الصارم**: كل جزء له مجلد خاص به
3. **التوثيق المستمر**: كل عملية تُسجل في التقارير
4. **نظام Checkpoint**: يمكن الاستئناف من أي نقطة

## تدفق العمل

```
المناجم (01) → عمال المناجم (02) → صناديق الذهب (03) → 
الكلاسات الذهبية (04) → الاختبار (05) → التقارير (06)
```

## قواعد التسمية

- المجلدات: أرقام + أسماء واضحة (01-mines)
- الملفات: snake_case (web_button_props.json)
- السكريبتات: descriptive_names (extract_radix_props.py)
