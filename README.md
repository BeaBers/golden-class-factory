# 🏭 Golden Class Factory

منصة تصميم ويب ثورية تعتمد على استخراج الخصائص من مكتبات مفتوحة المصدر 
وتحويلها إلى "كلاسات ذهبية" قابلة لإعادة الاستخدام.

## 📁 هيكل المشروع

- `01-mines/` - المناجم (المكتبات المصدرية)
- `02-miners/` - سكريبتات عمال المناجم
- `03-gold-boxes/` - صناديق الذهب (البيانات المستخرجة)
- `04-golden-classes/` - الكلاسات الذهبية
- `05-sandbox/` - أنبوب الاختبار
- `06-reports/` - التقارير
- `07-documentation/` - التوثيق
- `08-assets/` - الأصول العامة
- `09-backup/` - النسخ الاحتياطية

## 🚀 البدء

1. تأكد من تثبيت Git و Python 3.x و Node.js
2. شغّل سكريبت الإعداد (سيتم إنشاؤه لاحقاً)
3. راجع `state.json` لمتابعة التقدم

## 📊 متابعة التقدم

افتح `state.json` لرؤية الحالة الحالية للمشروع.
